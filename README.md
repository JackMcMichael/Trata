# Pomodoro Particle Timer
 
A full-screen Pomodoro timer for Android. No UI chrome, no distractions — just a black screen slowly filling with falling particles over 25 minutes.
 
## How it works
 
Particles fall from the top of the screen, column by column, left to right. The first particle falls almost immediately; the last settles at exactly the 25-minute mark. When the screen is full, **Break time** appears.
 
Controls are minimal: **Start**, **Pause**, and **Reset**.
 
## Install on Android
 
1. Download `pomodoro.html`
2. Transfer it to your Android phone
3. Open it in **Chrome**
4. Tap **⋮ → Add to Home screen**
It launches full-screen with no browser UI, behaving like a native app.
